# Quick Start - Icarus Server Control

## 🚀 Get Started in 3 Steps

### 1️⃣ Install Tailscale
Download from: https://tailscale.com/download

### 2️⃣ Login to Tailscale
- Email: `404@meduseld.io`
- Password: (ask admin)

### 3️⃣ Access the Panel
1. Open browser to: https://meduseld.io
2. Click "Enter the Great Hall"
3. Click "Open Control Panel"

**Direct link:** https://panel.meduseld.io

---

## 🎮 Control the Server

| Button | What it does | When to use |
|--------|--------------|-------------|
| 🟢 **Start** | Starts the server | When server is offline |
| 🔴 **Stop** | Stops the server | Normal shutdown |
| 🟡 **Restart** | Updates & restarts | Weekly maintenance |
| ⚫ **Kill** | Force stop | Emergency only! |

---

## ❓ Can't Access?

1. Check Tailscale is connected (green icon)
2. Try reconnecting Tailscale
3. Restart Tailscale
4. Contact admin

---

## 📱 Use on Phone/Tablet?

Yes! Install Tailscale on your mobile device, login with the same account, then access meduseld.io in your mobile browser.
